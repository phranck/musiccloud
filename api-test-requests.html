18 curl-Befehle für lokales API-Token-Testing. Basis-URL: http://localhost:4000

Token nicht gesetzt — authentifizierte Requests (6–10, 13, 18) enthalten Platzhalter.

Vorbereitung
  1. ./app start — Backend + Developer-App starten
  2. http://localhost:4321 — Developer-Account anlegen
  3. API Access beantragen, Client anlegen, Token generieren
  4. Token oben in die Leiste einfügen — wird automatisch in alle Requests eingesetzt
